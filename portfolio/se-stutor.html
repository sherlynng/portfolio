Feb 2018 - Apr 2018

About the Project

STUtor is a desktop app tailored for tuition coordinators to easily match and manage a database of tutors and students. This app is optimized for those who prefer to work with a Command Line Interface (CLI) while still having the benefits of a Graphical User Interface (GUI).

This project was done in a group of 4 members, as part of CS2103T module. We modified the existing Address Book application provided and transformed it into a tution coordinating application.

I worked on Remark, Rating and Autofill features in STUtor. View My Contributions
