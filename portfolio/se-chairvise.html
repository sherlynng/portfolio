About the Project

ChairVisE is a single page application used by conference program chair to visualize conference data. It receives input data files in .csv format and return visualizations drawn with the parsed data. The userâ€™s objective is to obtain meaningful insights and be able to access and export the visualizations easily, with an easy-to-use and intuitive user interface.

This project was done in a group of 4 members, as part of CS3219 module. Our roles was to improve the existing version of the application provided. I worked as a frontend developer, together with another team member.

Following are the enhancements made to the existing version:
  • Make existing visualizations more meaningful for the user.
  • Add more visualisations to provide more value to the user.
  • Improve the user interface, make system more useful and appealing.
  • Creating visualizations by joining multiple csv files.
  • User Authentication & Session Management.
  • Mapping data schemes.
    • The existing application retrieves data using column names. Therefore, another file with different column names cannot be used. To improve this, we used Vue.Draggable to allow users to reorder the columns. Hence, user does not have to manually change the file, making this approach more user-friendly.
